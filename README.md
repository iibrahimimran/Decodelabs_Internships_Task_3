# Project 3 – SQLite Database Analysis

## Overview

This project demonstrates the use of SQLite databases with Python for storing, querying, and analyzing structured data.

## Objectives

* Create an SQLite database
* Import data into database tables
* Execute SQL queries
* Perform data analysis using SQL
* Extract meaningful business insights

## Technologies Used

* Python
* SQLite (sqlite3)
* Pandas
* Jupyter Notebook

## Key Tasks

* Database creation
* Data insertion
* SQL querying
* Aggregation and filtering
* Data retrieval and reporting

## Outcome

The project showcases practical SQL skills and database management techniques for data analytics workflows.
